# Telegram AI Moderation Bot

Telegram-бот на **aiogram 3.x** с AI-модерацией через **Gemini API**.

## Возможности

- **Доступ в ЛС**: только владелец (`OWNER_ID`) получает полный доступ; остальным — «Не лезь, бот не для вас»; спам → бан на 7 дней
- **AI-модерация в группах**: анализ сообщений по правилам чата + этическим нормам
- **Ротация моделей Gemini**: от умной к простой; при исчерпании — следующий проект (2 API-ключа)
- **Очередь запросов**: асинхронная обработка, лимит 15 RPM
- **Батчинг**: по умолчанию 30 сек (настраивается, `0` = каждое сообщение отдельно)
- **Наказания**: мут, устное предупреждение, кнопка «Снять наказание»
- **Команды**: `/punishments`, `/setrules`, `/setinterval`, `/mod`, `/admin`

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Заполните BOT_TOKEN и GEMINI_API_KEY_1/2 в .env
python -m bot.main
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен Telegram-бота от @BotFather |
| `OWNER_ID` | Telegram ID владельца (по умолчанию 1776244625) |
| `GEMINI_API_KEY_1` | API-ключ первого GCP-проекта |
| `GEMINI_API_KEY_2` | API-ключ второго GCP-проекта |
| `RPM_LIMIT` | Запросов в минуту (по умолчанию 15) |
| `RPD_PER_MODEL` | Лимит запросов/день на модель (по умолчанию 1000) |
| `DEFAULT_BATCH_INTERVAL` | Интервал батчинга в секундах (по умолчанию 30) |

## Настройка в группе

1. Добавьте бота в группу
2. Назначьте **администратором** с правом **ограничивать участников** (для мута)
3. Загрузите правила: `/setrules` → ответьте текстом или .txt файлом
4. Настройте интервал: `/setinterval 30` (или `0` для мгновенной обработки)
5. Вкл/выкл модерацию: `/mod on` / `/mod off`

## Команды

| Команда | Где | Описание |
|---|---|---|
| `/admin` | ЛС (владелец) | Панель с лимитами API |
| `/punishments` | Группа | Активные наказания чата |
| `/setrules` | Группа | Загрузка правил |
| `/setinterval N` | Группа | Интервал батчинга (сек) |
| `/mod on/off` | Группа | Вкл/выкл модерацию |

## Модели (порядок использования)

1. `gemini-2.5-pro`
2. `gemini-2.5-flash`
3. `gemini-2.5-flash-lite`
4. `gemini-3-flash-preview`
5. `gemini-3.1-flash-lite-preview`

## Ограничения

- Бот видит только сообщения, полученные **после добавления в чат** (ограничение Telegram Bot API)
- Для мута нужны права администратора у бота
- API-ключи **не коммитьте** в репозиторий — только через `.env`
